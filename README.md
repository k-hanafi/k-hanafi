
Hello here's a TLDR overview of my work. Recently it's been about designing LLM systems optimizing cost and accuracy at scale.

***AI Research @ UBC***

- **[ai-adoption-deep-research-agent](https://github.com/k-hanafi/ai-adoption-deep-research-agent)** — Agentic deep research system optimizing Perplexity's API cost 8x and producing the largest academic-grade dataset of enterprise internal AI-adoption.
- **[ai-startups-taxonomy-research](https://github.com/k-hanafi/ai-startups-taxonomy-research)** — Classifying the entire Crunchbase repository into an ai-native taxonomy for startups.
- **[LLM-directness-experiment](https://github.com/k-hanafi/LLM-directness-experiment)** — A data contamination audit measuring pretraining bias in LLM classifications using feature ablation and chance-corrected agreement statistical tests.

***Projects***

- **[oai-batchkit](https://github.com/k-hanafi/oai-batchkit)** — A platform making batch inference cheaper and easier for researchers converting large-scale data into EDA-ready output with LLM's. Grew out of being a worldwide top user of OpenAI's Batch Processing API.

- **[NYC-airbnb-popularity-predictor](https://github.com/k-hanafi/NYC-airbnb-popularity-predictor)** — Applying Sklearn's tree-based ensemble models.
- **[SauderSuitsUp](https://saudersuitsup.com/)** — E-comm platform built on Shopify providing formalwear access for thousands of UBC students. $25k in inventory and $18k in funding raised so far.
